 Overview

The 11/11 IKS / QKS APIs provide cryptographic key governance and execution primitives.

These APIs expose controlled interfaces only.  
Internal cryptographic mechanisms, key material and policy engines are intentionally abstracted.

 Capabilities
- Secure key lifecycle operations
- Policy-governed execution
- Auditable request/response flows
